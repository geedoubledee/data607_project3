Description:
The Client maintains various computer systems that are used in all the courts and other
state agencies throughout the client. Client is seeking a Business Analyst for development of a central scheduling system, along with enhancement, integration, and documentation for changes to the eCourt applications. The eCourt system is comprised of Windows and web based .Net applications with a SQL Server 2016 database backend. A major function of the system is to integrate the audio that is recorded in courts through FTR with the log notes that are entered during these court sessions. More features will be added to these systems to allow for more robust backup and integration with the existing application as we move forward.
