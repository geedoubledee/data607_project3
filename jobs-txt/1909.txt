Job Summary:
The Sr. Business Intelligence Developer should have experience in utilizing Azure Stack for data analytics platforms (Data Factory, Data Lake, Azure Synapse Analytics, Machine Learning, SQL Warehouse, Power BI, Azure Analysis Services) and maintain our current On-Prem Microsoft SQL Server, Integration Services (SSIS), SQL Server Reporting Services (SSRS) and SQL Server Analysis Services (SSAS) systems until we fully migrate to Azure.
The Sr. BI Developer will design, implement, and maintain the Extraction, Transformation, and Loading (ETL) processes using Structured Query Language (SQL) Server, Data factory, SSIS to support the successful creation of operational reports and interactive dashboards used for our company.
The Sr. Business Intelligence Developer will also mentor and guide other developers within the team in the best practices of software development.
