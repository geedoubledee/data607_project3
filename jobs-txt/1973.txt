Junior Oracle Database Administrator - 100% Remote!
Exciting opportunity for recent college grads!
GDIT is immediately hiring for a Junior Oracle Database Administrator. The Junior Oracle DBA will administer and maintain the security and integrity of the databases, applications in addition to providing incident management and problem support.
The Junior Oracle DBA will work under the leadership of a Sr. Oracle DBA on Automation tools, monitoring and also be responsible for database capacity issues, replication and overall performance issues, Oracle SR Support follow-ups and readme document analysis. This is a great opportunity to work with the latest version of Oracle 19c and with hosting and cloud environment.
What Junior DBA's Do:
Monitor database performance, implement changes and apply new patches and upgrades, when required. Monitor and optimize database and server performance
Design, create, and assist data modeling and database schema in accordance with business requirements.
Create complex functions, scripts, stored procedures and triggers to support application development
Develop best practices for database design, development activities, troubleshooting and root cause analysis.
Perform regular tests and evaluations to ensure data backup, security, privacy, and integrity
Test and implement database backup and recovery operations.
Maintain existing stored procedures and review existing SQL code
Create and maintain database documentation, including data standards, procedures, and definitions for the data dictionary (metadata).
Monitor alert system details and reporting.
Proactively and regularly make recommendations for system improvements.
Perform business intelligence and business performance management activities.
